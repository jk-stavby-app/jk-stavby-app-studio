# JK Stavby Dashboard

Interní aplikace pro správu stavebních projektů a financí.

![Author](https://github.com/jk-stavby-app/jk-stavby-app-studio/blob/main/Daniel-vilim-one.webp?raw=true)

## Spuštění
```bash
npm install
npm run dev
```

## Deployment (Vercel)

1. Push na GitHub
2. Import do Vercel
3. Nastav environment variables:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
4. Deploy

## Tech Stack

React · TypeScript · Tailwind CSS · Vite · Supabase · Recharts
